# State & Tribal Toxin Thresholds and HAB Advisory Decision Methods | US EPA

**What it is.** An EPA webpage that compiles, state by state and tribe by tribe, the numeric cyanotoxin thresholds (for anatoxin-a, microcystins, cylindrospermopsin, and saxitoxins, all in µg/L) that U.S. state and tribal programs actually use to trigger drinking-water and recreational-water HAB advisories, with a link to each jurisdiction's own source guidance document. It is a curated regulatory/decision-method reference, not an original research study.

## Methods
Not an empirical study — this is a manually curated policy/regulatory reference table. EPA staff located each U.S. state's and select tribal nations' publicly published cyanotoxin advisory guidance (for anatoxin-a, microcystins, cylindrospermopsin, saxitoxins) and tabulated the numeric drinking-water and recreational-water action levels into one comparison table, hyperlinking to each jurisdiction's original source document. Where a jurisdiction had not published an independent numeric value, the table records that it defaults to EPA's own Health Advisory (drinking water) or Ambient Water Quality Criteria (recreational) values, which are themselves listed as reference baselines. There is no fieldwork, sampling, statistical modeling, or hypothesis testing involved; the "method" is documentary compilation and linking, and it functions purely as a look-up/reference resource. It "works" as a single-stop comparison across ~50 states/6 tribes with traceable source links; it explicitly does not work (i.e., leaves gaps) wherever no public threshold document could be found, and it does not reconcile or validate why different jurisdictions chose different numeric cut points for the same toxin and use case.

## Stated limitations
The page states directly that "only thresholds for which there was publicly accessible information available are listed," meaning blank/"No Threshold" cells reflect a documentation-availability gap, not necessarily the absence of any state policy. It also states that "many states used" EPA's own HA or AWQC values rather than independently derived numbers, so the apparent breadth of "50-state" coverage substantially reflects adoption of one federal default rather than 50 independently validated thresholds. A second, targeted check of the page found no on-page definition of what EPA HA/AWQC methodology entails (readers must follow outbound links to the underlying EPA or state documents) and no explicit disclaimer about the completeness, accuracy, or update-currency of the compiled state values relative to each state's actual current regulations. The resource carries a "Last updated on June 25, 2026" date but provides no versioning or accuracy guarantee beyond that timestamp, so entries for any state that has since revised its guidance could be stale. As a compiled regulatory-reference table rather than a scientific study, it reports no statistical uncertainty, error bars, or validation of whether any given threshold is health-protective in practice.

## Tensions with other findings
This source documents, rather than resolves, wide cross-jurisdiction inconsistency: for the identical toxin and use case (e.g., recreational microcystin exposure), action thresholds range from as low as a 1 µg/L "caution" trigger in California to single "warning" bands stretching from about 8 µg/L up to 2,000 µg/L in Nevada/Kansas/Utah, and some states have no numeric threshold at all. Any HAB analytics or risk-communication tool that reports a single "advisory" or "action" level therefore cannot claim one universal, nationally validated numeric truth — it must specify and disclose which jurisdiction's (or EPA's own default) framework it is emulating. Separately, every threshold in this compilation is a toxin-concentration value (µg/L); it contains no cyanobacterial cell-count, biovolume, or chlorophyll-a based alert level, which is a different action-threshold metric used elsewhere in HAB advisory literature (e.g., WHO-style cell-density alert frameworks). A tool built on remote-sensing/cell-density or chlorophyll-a proxies would need an additional, separately-sourced and separately-uncertain toxin-to-biomass conversion step before it could be compared against any of the µg/L values in this source. Finally, because many "state" entries are simply EPA's own HA/AWQC value re-labeled by state, apparent inter-state consensus in this table partly reflects administrative adoption of one federal number rather than convergent independent scientific derivation — a caution against treating "N states agree" counts, drawn from this or similar compilations, as independent corroboration.
