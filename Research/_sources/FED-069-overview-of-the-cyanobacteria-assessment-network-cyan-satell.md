# Overview of the Cyanobacteria Assessment Network (CyAN) satellite monitoring and the future of expanding to >270,000 lakes

**What it is.** An EPA Science Inventory record for the abstract of a 2023 NALMS conference presentation by EPA scientists (Schaeffer, Guenzel, Salls) giving a program-level overview of the Cyanobacteria Assessment Network (CyAN) satellite HAB monitoring effort, its current validation basis and public data products, and EPA's plans to expand CyAN's coverage and capabilities.

## Methods
The abstract describes no specific method, sensor, or algorithm name (e.g., it does not name Sentinel-3/OLCI or a specific cyanobacteria index within this text) — it only states, at a program level, that "the satellite algorithm used in CyAN has been validated with Water Quality Portal chlorophyll-a, state reported advisories, and state reported cell counts and toxins." That is: three categories of in-situ/state reference data (WQP chlorophyll-a measurements, state-issued bloom advisories, and state-reported cyanobacteria cell counts/toxin concentrations) are named as the validation basis, but no accuracy statistics, sample sizes, or comparison results are given in this record. Data products described as outputs: CyAN mobile and web apps, EPA's How's My Waterway tool, quantified metrics served via EnviroAtlas and EPA's Report on Environment, and GeoTIFF files via direct download.

## Stated limitations
The abstract itself flags a limitation/gap as forward work: "the need for discrete field validation information" to support expanding CyAN's chlorophyll-a detection and coverage to over 270,000 lakes — implying that current field validation is not considered sufficient for that expanded scope. Beyond that, this record provides no quantitative validation statistics, no case-study details, and no methodological specifics (it is a conference abstract, not a methods/results paper), which limits what can be verified or reused from this source alone.

## Tensions with other findings
This is a program overview/roadmap abstract rather than a peer-reviewed validation study: it asserts that "case studies demonstrate CyAN satellite data helps save resources and protect human health" with no supporting data in this record, immediately followed by an acknowledgment that further "discrete field validation information" is still needed for the planned expansion — i.e., a confident outcome claim and an admitted validation gap appear side by side in the same short abstract. It should be treated as a scope/roadmap citation (current ~2,000-lake coverage vs. a >270,000-lake aspiration, and the three-part validation basis) rather than as a source of independent accuracy figures when reconciled against more quantitative CyAN algorithm-validation papers elsewhere in the review.
